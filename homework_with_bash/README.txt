Изменения переменной PATH через команду export являются временными и относятся к текущей сессии. После закрытия терминала изменения не сохраняются.
Для сохранения изменений export нужно добавить в файл конфигурации. Например, .bashrc.

nano ~/.bashrc
export PATH="$PATH:/c/Users/tujko/my_scripts"
echo "$PATH"
